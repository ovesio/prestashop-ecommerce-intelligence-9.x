# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it responsibly.

### Preferred contact (anti-spam format)
security [at] ovesio [dot] com

### Alternative contact
You can also use our contact form:
https://ovesio.com/contact

Please include:
- module name and version
- affected platform (PrestaShop / Magento / OpenCart / WordPress / WooCommerce, etc...)
- steps to reproduce
- expected vs actual behavior
- proof-of-concept (if available)

## Please do not disclose security issues publicly
Do not open public GitHub issues for security reports.
